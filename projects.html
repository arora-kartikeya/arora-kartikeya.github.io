Quantum Computing Libraries

QuantumVQE

A comprehensive Python library for variational quantum eigensolvers with advanced optimization techniques and noise mitigation strategies. Designed for both research and educational purposes.

245 stars 42 forks 1.2k downloads

Key Features:

  • Support for multiple ansatz types (UCCSD, Hardware Efficient, etc.)
  • Advanced optimization algorithms (SPSA, COBYLA, L-BFGS-B)
  • Built-in noise mitigation techniques
  • Extensive documentation and tutorials
  • Integration with Qiskit, Cirq, and PennyLane

QEC-Simulator

High-performance quantum error correction simulator with support for various code families and noise models. Optimized for large-scale simulations and research applications.

189 stars 28 forks 850 downloads

Supported Error Correction Codes:

  • Surface codes (rotated and planar)
  • Color codes (triangular and square lattice)
  • Repetition codes
  • CSS codes (Steane, Shor, etc.)
  • Custom stabilizer codes
C++ Python OpenMP PyBind11 NetworkX
